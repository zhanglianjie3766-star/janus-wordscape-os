# 设置页交互冻结文档 v1.0

本文档冻结“雅努斯词境 OS / TechLex OS”底栏菜单 `设置` 页的功能范围、信息架构、折叠规则、核心模块和 UI 设计。

设置页是系统控制台，不是学习页。它用于配置规则、导入备份、检查数据健康、管理个人资料和阅读必要说明。

## 1. 页面定位

设置页只回答三个问题：

```text
我的学习规则是什么？
我的词卡和本地数据是否安全？
我如何理解应用、FSRS 和基本操作？
```

设置页不得承载：

```text
正式复习卡片
单词列表浏览
图谱交互
统计仪表盘
```

## 2. 信息架构

设置页分三组：

```text
常用
说明
系统
```

模块顺序冻结为：

```text
常用：
1. 学习规则
2. 个人资料
3. 词卡与备份
4. 数据健康

说明：
5. FSRS 记忆算法
6. 关于应用
7. 操作手册

系统：
8. 高级操作
```

## 3. 折叠规则

所有模块默认折叠。

冻结规则：

```text
初次进入设置页，不自动展开任何模块。
点击一个模块，展开该模块。
点击另一个模块，上一个模块自动折叠。
再次点击当前展开模块，折叠回去。
同一时间最多一个模块展开。
允许全部模块都处于折叠状态。
```

不得恢复：

```text
默认展开学习规则
多个模块同时展开
进入设置页后页面很长
```

## 4. 模块卡片视觉

每个设置模块使用紧凑卡片。

卡片结构：

```text
左侧圆形图标
中部标题
中部一行摘要
右侧箭头
```

视觉要求：

```text
卡片高度紧凑。
摘要最多一行，超出截断。
图标低噪声灰色，展开时使用主色浅底。
展开状态边框轻微使用主色。
```

## 5. 学习规则

学习规则模块用于配置每日学习和复习行为。

字段冻结：

```text
每天新学数量
每天复习上限
每天弱项巩固上限
目标保持率
最大复习间隔（天）
遗忘后重学间隔（分钟）
记忆难点阈值（Again 次数）
复习排序
暂停新卡
弱项巩固
逾期优先
```

保存规则：

```text
修改字段后必须点击保存学习规则。
保存后更新 learning_plan。
不得自动清空 ReviewEvent 或 UserMemoryState。
```

## 6. 个人资料

个人资料用于本地显示。

字段冻结：

```text
昵称
头像
```

默认值：

```text
昵称：JanusAI
头像：默认 AI 头像
```

规则：

```text
支持上传本地头像图片。
头像建议小于 512KB。
昵称为空时恢复默认昵称。
个人资料只影响 UI 显示，不影响词卡、复习、统计和图谱数据。
```

## 7. 词卡与备份

词卡与备份是数据进出入口。

功能冻结：

```text
选择文件导入
导入核心验收包
导入演示包
导入小样例
导出备份
恢复备份
导入前快照
最近导入记录
导入错误报告
导入提醒
```

导入规则：

```text
支持 JSON / CSV / TSV。
导入前校验字段。
导入前保存快照。
重复词卡按策略跳过。
导入失败不得破坏当前学习数据。
```

备份规则：

```text
导出为 TechLex OS JSON 备份。
恢复备份需要选择 JSON 文件。
导入前快照只恢复最近一次导入前状态。
```

## 8. 数据健康

数据健康用于检查当前本地数据状态。

展示内容：

```text
完整性状态
存储大小
错误 / 提醒数量
完整性问题列表
```

规则：

```text
数据健康只读。
不得在本模块直接清空数据。
发现问题时给出问题列表。
无问题时显示正常提示。
```

## 9. FSRS 记忆算法

本模块用于解释智能调度和版权合规。

必须说明两层机制：

```text
FSRS 正式评分
浏览触发式遗忘标记
```

### 9.1 FSRS 正式评分

说明要点：

```text
记忆卡显示答案后，忘记 / 困难 / 良好 / 简单 是 FSRS 的核心输入。
FSRS 根据评分动态调整下次复习间隔。
正式评分写入 review_mode=daily_task。
```

### 9.2 浏览触发式遗忘标记

说明要点：

```text
点击单词列表词卡表示不认识。
系统写入 browser_detail + again。
该词卡进入今日待复习。
这不是 FSRS 原始算法规则，而是产品 Adapter / Queue Layer 规则。
```

### 9.3 协同关系

说明要点：

```text
浏览触发式遗忘标记负责发现陌生词。
FSRS 负责正式复习后的间隔调度。
browser_detail 不计入正式已复习。
daily_task 才计入正式复习统计。
```

### 9.4 版权声明

必须保留：

```text
ts-fsrs 5.4.0
MIT License
Copyright (c) 2026 Open Spaced Repetition
```

## 10. 关于应用

关于应用用于说明产品定位和基础信息。

内容冻结：

```text
定位：雅努斯词境 OS 是面向 AI 编程英语的个人词境记忆系统。
核心路径：按真实技术场景整理词卡，用 FSRS 安排复习，用图谱诊断为什么记不住。
数据方式：词卡、记忆状态和复习记录优先保存在本机 IndexedDB，并支持导出和恢复备份。
开发者：雅努斯（Janus）
应用版本
备份版本
```

不得把“开发者”误写成用户字段。

## 11. 操作手册

操作手册只保留最小必要路径。

步骤冻结：

```text
1. 导入词卡
2. 学习与浏览
3. 今日复习
4. 统计与图谱
```

每一步只写核心动作，不写长教程。

## 12. 高级操作

高级操作用于低频危险操作。

展示内容：

```text
当前存储来源
本地存储状态
清空本地数据
```

清空规则：

```text
必须弹出确认。
用户确认后才允许执行 resetData。
清空操作不得误放在常用模块。
```

## 13. 与其他页面关系

设置页对其他页面的影响：

```text
学习规则 -> 影响今日页队列、单词本内联学习、FSRS 预览间隔。
词卡导入 -> 影响单词本、今日页、统计页、图谱页。
个人资料 -> 影响侧栏头像昵称，不影响学习数据。
数据健康 -> 只读检查。
高级清空 -> 重置所有本地学习数据。
```

## 14. 不得回退

以下行为不得重新引入：

```text
设置页默认展开学习规则。
设置页所有模块同时展开。
设置页变成长表单页面。
导入失败仍修改当前数据。
FSRS 说明里把浏览触发式遗忘标记伪装成 FSRS 原始算法。
关于应用里把开发者写成用户。
高级清空无需确认。
个人资料影响词卡或记忆状态。
```

## 15. 回归验收清单

每次改设置页后至少检查：

```text
进入设置页，所有模块默认折叠。
点击学习规则展开，再点击个人资料时学习规则自动折叠。
再次点击当前模块可折叠。
学习规则保存后 learning_plan 更新。
个人资料可保存昵称和头像。
词卡与备份可选择文件导入。
导入失败不破坏当前数据。
数据健康显示完整性和存储大小。
FSRS 记忆算法说明包含 ts-fsrs 5.4.0、MIT、FSRS 与浏览触发式遗忘标记的区别。
关于应用显示开发者：雅努斯（Janus）。
高级操作清空本地数据需要确认。
手机端所有模块不横向溢出。
```
