# UI 回归脚本冻结文档 v1.0

本文档冻结“雅努斯词境 OS / TechLex OS”五个底栏页面的 UI 回归脚本目标、设备矩阵、关键断言和禁止行为。

它不是一次性测试计划，而是未来从 0 复刻或重构时必须保留的回归标准。

## 1. 回归目标

回归脚本必须覆盖：

```text
今日
单词本
统计
图谱
设置
```

目标：

```text
确认页面结构未漂移。
确认移动端不横向溢出。
确认核心交互能完成。
确认数据口径没有把 browser_detail 与 daily_task 混淆。
确认图谱可选中、可二次打开、可拖动、可缩放、可刷新。
```

## 2. 测试环境边界

默认不得清空当前 in-app browser 的 IndexedDB/localStorage。

需要清库或导入固定数据时，必须使用隔离 browser context。

推荐命令：

```bash
corepack pnpm run typecheck
corepack pnpm run build
```

如果使用 Playwright：

```text
使用独立 context。
使用固定测试包。
测试完成后关闭 context。
不得影响用户真实浏览器数据。
```

## 3. 设备矩阵

移动端：

```text
360 x 780
390 x 844
412 x 915
430 x 932
```

平板：

```text
768 x 1024
```

桌面：

```text
1280 x 720
1440 x 900
```

每个 viewport 必须断言：

```text
body scrollWidth <= viewport width + 1
底栏可见
主内容未被底栏遮挡关键按钮
无控制台 error
```

## 4. 固定测试数据

回归数据应至少包含：

```text
2 个领域包
每个领域包 2 个二级场景
每个二级场景 5 张词卡
至少 1 张未学习
至少 1 张学习中
至少 1 张待复习
至少 1 张已掌握
至少 1 张带音标和读音
至少 1 张带词族、易混词、标签、来源
```

ReviewEvent 必须包含：

```text
daily_task again
daily_task good
browser_detail again
```

用于验证统计边界。

## 5. 今日页脚本断言

步骤：

```text
进入 今日。
确认顶部只有 待复习 / 已复习 / 剩余。
点击 待复习 问号。
确认说明弹层出现并可关闭。
如果有待复习卡，点击 显示答案。
确认四个评分按钮在答案详情下方。
点击 良好。
确认进入下一张或完成态。
```

断言：

```text
显示答案前不出现 来源 / 下次到期 / 复习次数。
评分写入 daily_task。
今日已复习增加。
browser_detail 不增加今日已复习。
```

## 6. 单词本主页面脚本断言

步骤：

```text
进入 单词本。
点击 全部 领域 chip。
确认显示二级场景文件夹，不是平铺词卡列表。
确认场景卡片整张可点击。
点击一个场景卡片。
```

断言：

```text
顶部状态统计为 未学习 / 学习中 / 待复习。
场景卡片显示 未学习 / 学习中 / 待复习 / 已掌握。
场景卡片无 开始学习 按钮。
场景卡片无 单词列表 按钮。
场景卡片无粗进度横线。
```

## 7. 单词本子页脚本断言

步骤：

```text
进入二级场景子页。
确认标题居中。
确认左侧只有返回图标。
确认右侧只有设置图标。
确认状态筛选为 全部 / 未学习 / 学习中 / 待复习 / 已掌握，并显示数字。
点击设置图标。
选择词频。
选择一级和二级场景。
关闭弹层。
点击一张词卡行。
```

断言：

```text
词卡范围弹层不横向溢出。
词频选项移动端两列排布。
场景为单选级联。
点击读音按钮不触发 browser_detail。
点击词卡行触发 browser_detail + again。
该词卡进入今日待复习。
状态 badge 可见。
```

内联学习断言：

```text
点击开始学习后不跳转今日页。
显示内联学习卡。
显示答案后出现四个评分。
点击评分写入 daily_task。
```

## 8. 统计页脚本断言

步骤：

```text
进入 统计。
检查四个仪表盘顺序。
逐个点击问号。
```

断言：

```text
没有顶部“统计”标题。
仪表盘顺序为 今日执行 / 记忆健康 / 阶段分布 / 薄弱场景 Top 3。
每次只打开一个说明弹层。
弹层可关闭。
数字不过粗、不过大。
手机端四个仪表盘间距紧凑。
薄弱场景列表没有厚重嵌套边框。
browser_detail 不计入 今日已复习 和 Again率。
```

## 9. 图谱页脚本断言

步骤：

```text
进入 图谱。
确认标题不与节点重叠。
鼠标或触摸第一次接触节点。
确认节点高亮但不打开详情。
再次点击同一节点。
确认打开详情。
点击关闭按钮。
再次点击同一节点可关闭详情。
点击左上图谱图标。
```

断言：

```text
第一次高亮后，移动鼠标不再点亮其他节点。
拖动画面不打开详情。
拖动节点时可带动局部布局。
鼠标滚轮可缩放。
双指触控可缩放。
左上图谱图标可清除选中并恢复未选中状态。
详情弹层不遮挡右上设置和铅笔按钮。
默认灰色节点在手机端可见。
```

## 10. 设置页脚本断言

步骤：

```text
进入 设置。
确认所有模块默认折叠。
展开学习规则。
展开个人资料。
展开 FSRS 记忆算法。
展开关于应用。
展开操作手册。
```

断言：

```text
展开一个模块时，其他模块自动折叠。
再次点击当前模块可折叠。
FSRS 说明包含 ts-fsrs 5.4.0 和 MIT。
FSRS 说明区分正式评分与浏览触发式遗忘标记。
关于应用显示开发者：雅努斯（Janus）。
设置页不横向溢出。
```

## 11. 数据联动脚本断言

必须验证一条完整链路：

```text
单词本子页点击词卡行
-> 写入 browser_detail + again
-> 词卡进入今日待复习
-> 今日待复习增加
-> 今日已复习不增加
-> 统计 Again率不增加
-> 在今日页正式评分
-> 写入 daily_task
-> 今日已复习增加
-> 统计 Again率按 daily_task 更新
```

PC 和移动端均要执行。

## 13. 禁止回退清单

```text
测试脚本清空用户真实 IndexedDB/localStorage。
只测桌面不测手机。
只看截图不测 ReviewEvent。
只测 daily_task 不测 browser_detail。
图谱只测打开不测高亮/二次点击/拖动/缩放。
设置页默认展开学习规则。
统计页仪表盘间距过大。
```
