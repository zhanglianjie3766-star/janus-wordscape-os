# 组件规格冻结文档 v1.0

本文档冻结“雅努斯词境 OS / TechLex OS”五个底栏页面共用组件的结构、行为和视觉约束。

页面冻结文档负责页面语义；本文档负责组件复刻。

## 1. 底栏导航

底栏固定五项：

```text
今日
单词本
统计
图谱
设置
```

规则：

```text
移动端固定在底部。
PC 端可使用左侧栏，但语义和顺序保持一致。
选中项使用 brand-soft 背景和 brand 图标/文字。
未选中项使用 muted 图标/文字。
底栏不得遮挡页面关键按钮。
```

## 2. 页面统计卡

用于今日页、单词本主页面、统计仪表盘。

结构：

```text
白底整体卡片
内部横向指标
指标名在上，数字在下
弱分隔线只在必要时使用
```

规则：

```text
指标字段多于 3 个时，优先压缩字号和间距，不强行换行。
统计数字使用 tabular-nums。
数字不得使用 700 以上字重。
移动端保持一行横向排布。
```

## 3. 状态筛选条

用于单词本二级场景子页。

固定字段：

```text
全部
未学习
学习中
待复习
已掌握
```

规则：

```text
每项显示中文标签和数字。
选中态使用 brand 背景和白字。
未选中态使用透明或浅灰底，文字 muted。
一行优先，窄屏允许横向滚动，不允许挤压错行。
```

交互：

```text
点击切换筛选。
切换筛选不离开当前子页。
筛选结果实时影响下方开始学习按钮、词卡列表和范围小字。
```

## 4. 领域包 Chip 横向入口

用于单词本主页面。

结构：

```text
全部
AI编程
Web3
语言运行时
AI平台
...
```

规则：

```text
横向滚动。
选中态 brand。
每个 chip 显示词卡数量。
点击全部进入跨领域二级场景文件夹视图。
点击领域包进入该领域包下的二级场景文件夹视图。
```

不得点击 `全部` 后展示平铺单词列表。

## 5. 二级场景卡片

用于单词本主页面。

结构：

```text
左：文件夹圆形图标
中：二级场景标题
中下：未学习 / 学习中 / 待复习 / 已掌握
右：掌握百分比 + 右箭头
```

规则：

```text
整张卡片可点击进入子页。
百分比和箭头只是暗示入口。
不放开始学习按钮。
不放单词列表按钮。
不放粗进度横线。
不放外层大背景容器。
```

标题：

```text
字体 20-24px。
字重 700。
颜色 ink。
移动端必须不遮挡右侧百分比和箭头。
```

状态统计：

```text
四个状态必须一行优先。
如果空间不足，缩小字体或压缩间距，不拆成多行错位。
```

## 6. 二级场景子页头部

结构：

```text
左：圆形返回图标
中：场景/范围标题
右：设置图标
下：状态筛选条
下：范围说明小字
```

规则：

```text
返回图标旁不显示“场景列表”等重复文案。
右上不显示词卡总数文本。
标题居中、加粗、黑色。
设置图标打开词卡范围弹层。
```

范围说明：

```text
范围：{场景} · {状态} · {词频} · {N 张}
```

范围说明应与状态筛选条左侧视觉对齐。

## 7. 词卡范围弹层

结构：

```text
标题：词卡范围
右上：关闭
选择词频
选择场景
```

词频：

```text
全部
F1高
F2中高
F3中
F4低
```

规则：

```text
词频支持多选。
选择全部时清空其他词频。
词频选项移动端两列优先，宽度与场景按钮协调。
```

场景：

```text
左列：全部 + 一级场景
右列：全部 + 当前一级下二级场景
```

规则：

```text
场景为单选。
一级和二级级联。
用户界面只显示“全部”，不显示“一级全部/二级全部”。
一级场景名称不超过 5 个汉字视觉宽度。
二级场景名称不超过 7 个汉字视觉宽度。
超长名称使用省略，不撑破布局。
```

## 8. 词典式词卡行

用于单词本子页词卡列表。

结构：

```text
左：读音图标
中上：英文单词
中下：中文释义摘要
右：状态 badge + 右箭头
```

规则：

```text
点击读音图标只播放声音，不触发“不认识”写入。
点击行主体视为打开详情。
首次打开详情触发产品规则：不认识，写入 browser_detail + again，并加入今日待复习。
状态 badge 必须可见。
状态 badge 使用轻量状态色，不使用过强色块。
```

排序：

```text
默认按导入或生成顺序倒序。
未来有 created_at 字段时改为 created_at desc。
```

## 9. 词卡详情展开

用于单词本子页列表详情。

必须显示：

```text
词频
记忆状态
场景标签
音标和读音
中文释义
英文释义
来源
例句
词族
易混词
同义词
标签
```

必须显示写入结果：

```text
浏览详情记忆写入
最近记录
```

demo 词卡不得写入真实 ReviewEvent。

## 10. 内联学习卡片

用于今日页和单词本子页内联学习。

显示答案前：

```text
顶部标签：词频 / 阶段 / 场景
问题提示
英文大词
读音胶囊
显示答案按钮
```

显示答案前不得展示：

```text
来源
下次到期
复习次数
FSRS 内部状态
```

显示答案后：

```text
中文释义
英文释义
例句
词性
词族
易混词
同义词
标签
四个评分按钮
```

评分按钮必须在答案详情下方，不得覆盖内容。

## 11. FSRS 评分按钮

固定四项：

```text
1 忘记
2 困难
3 良好
4 简单
```

规则：

```text
四项是 FSRS 正式评分输入。
点击后写入 review_mode=daily_task。
按钮使用轻量色块，不再套额外厚边框容器。
移动端不固定在底部，避免遮挡答案详情。
```

键盘：

```text
1 -> 忘记
2 -> 困难
3 -> 良好
4 -> 简单
```

## 12. 问号说明弹层

用于今日页和统计页。

规则：

```text
问号紧贴标题或字段。
每次只允许打开一个弹层。
点击其他问号时，当前弹层自动关闭。
弹层必须有关闭按钮。
弹层文字短句化，直击统计口径。
```

统计页弹层：

```text
前两个仪表盘弹层在问号下方。
后两个仪表盘弹层在问号上方。
移动端整体向左避让，不能超出右边屏幕。
```

## 13. 设置页折叠卡片

结构：

```text
图标圆
标题
摘要
箭头
展开内容
```

规则：

```text
默认全部折叠。
点击当前卡片可展开。
再次点击当前卡片可折叠。
展开一个卡片时，其他卡片自动折叠。
```

卡片高度必须紧凑，避免设置页显得过长。

## 14. 图谱控制按钮

图谱页面固定入口：

```text
左上图谱图标：刷新/清除选中/恢复未选中
右上齿轮：关系诊断
右上柔和笔杆：显示微调
```

不得增加独立刷新按钮。

按钮必须避开图谱主体，不应遮挡关键节点。

## 15. 禁止回退清单

```text
场景卡片重新塞入开始学习和单词列表按钮。
词卡列表状态 badge 消失。
评分按钮遮挡答案详情。
词卡范围弹层横向溢出手机屏幕。
设置页默认展开学习规则。
多个说明弹层同时打开。
图谱需要第三个按钮才能清除选中。
```
