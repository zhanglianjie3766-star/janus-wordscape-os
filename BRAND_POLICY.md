# Brand Policy

`雅努斯词境 OS`, `Janus Wordscape OS`, related names, logos, icons, screenshots,
and visual marks identify this project.

The AGPL software license and CC BY-SA documentation/content licenses do not
grant a trademark or brand license.

## Allowed

You may use the project name to truthfully refer to this project, for example:

- "compatible with Janus Wordscape OS";
- "forked from Janus Wordscape OS";
- "uses Janus Wordscape OS source code under AGPL".

## Not Allowed Without Written Permission

You may not:

- present a modified fork as the official project;
- use the project name or marks in a way that implies endorsement;
- remove copyright, license, or third-party notices from distributions;
- use confusingly similar names for a competing distribution.

## Fork Naming

Modified community versions should use a distinct name and clearly state their
relationship to the upstream project.

Suggested wording:

```text
This project is a fork of Janus Wordscape OS and is not an official Janus
release.
```
