# Changelog

All notable release changes are tracked here.

## v0.1.0-alpha - 2026-05-23

Initial GitHub-ready alpha baseline.

Included:

```text
local-first PWA runtime
standard word-card package import
IndexedDB persistence
backup export and restore
FSRS scheduling through ts-fsrs 5.4.0
auditable ReviewEvent scheduler_version
browser_detail unfamiliar-word product rule
five bottom-tab UI: Today, Notebook, Stats, Graph, Settings
technical-English vocabulary network sample package
GitHub CI and Pages deployment workflow
```

Known limits:

```text
no remote sync
no account system
no app-store package
no multi-device conflict resolution
production build may still exceed Webpack's default recommended bundle-size warning threshold
```
